I created a plugin for Übersicht. The plugin allows for a list of stocks that are defined to be shown on the desktop. The plugin will show the stocks current price, daily change in both point and percent form.

Stock Tracker uses the [IEX](https://iextrading.com/) API to provide up to date data. The colors of stocks will change according to how large the daily change is.

To find out more information and how to install it, check out [Stock Tracker](https://github.com/stefanrsathianathen/stockTracker).   
Make sure that you have installed [Übersicht](http://tracesof.net/uebersicht/) for it to work.
